# cgo-example
cgo example

reference [Using C Libraries From Go](https://www.thegoldfish.org/2019/04/using-c-libraries-from-go/)

### BUILD so
To build the person.so, use 
```bash
build-libperson.sh
```


### COMPILE c
To compile hello.c, use
```bash
build-helloc.sh
```


### RUN go
To Test go, use
```bash
go run hello1.go
go run hello2.go
```
